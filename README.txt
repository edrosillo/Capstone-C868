Title: Appointment Management System
Purpose of the application: The purpose of the Appointment Management System is for Company ABC to be able to manage their Appointments and Customers
from a single easy to use interface.
Author: Eduardo Rosillo
Contact information: email: erosil1@wgu.edu
Version: 0.8
Date: 02/05/2022
Built with: IntelliJ Community 2021.2.3, full JDK version 11.0.4, and JavaFX-SDK-11.0.3
MySQL Connector: mysql-connector-java-8.0.27
Directions for how to run the program:
- In order to run the program the user must have valid credentials.
- Once the user has been validated access will be granted to the main interface.
- From the main interface the user will be able to access the three main sections that make up the program.
  (Appointments, Customers, Reports)
- From the Appointments Menu the user will be able to create, update and delete appointments.
- In the Customers Menu the user has the ability to create,  update and delete customers.
- Finally in the Reports Menu the user can access reports for the total number of customer appointments by type and month.
  A schedule for each contact in the organization.
  And the last type of report available to the user is the ability to view all the Customers that are located within any given Division.
